# Mushroom Dataset – Modelling Project

This repository contains the work developed for the university course Data Processing and Analysis.

## Project Overview

The goal of this project is to build a statistical or machine learning model using the Mushroom dataset provided by the UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/73/mushroom

The objective is to create a model capable of making predictions and/or uncovering meaningful patterns within the data.

## Objectives
Select and justify an appropriate modelling approach, train a model using the dataset, evaluate model performance using suitable metrics and interpret the results and extract insights

## Technologies

The project is implemented in Python, using standard libraries for data analysis and machine learning.

## Methodology

The workflow followed in this project includes:

1. Data understanding and preprocessing

2. Model selection and justification

3. Model training

4. Performance evaluation

5. Result interpretation

## Evaluation

Model performance is assessed using appropriate metrics depending on the chosen approach (e.g., accuracy, precision, recall, etc.).

## Collaboration

This project is developed as a group activity (group by 4).
Team members collaborate by dividing tasks and contributing to different phases of the project, including data processing, modelling, evaluation, and documentation.
