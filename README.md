# Data Processing and Analysis

This repository contains materials developed for the university course Data Processing and Analysis.

## Contents
A collection of lab exercises completed during the lectures
An optional project developed as part of the course for grading purposes

## Technologies

The work in this repository is primarily implemented in Python, using common libraries for data processing and analysis.

## Purpose

The goal of this repository is to showcase practical applications of data processing techniques, including data manipulation, analysis, and basic exploration workflows.

## Notes

This repository is intended for educational purposes and may include exploratory code, exercises, and project work developed throughout the course.
